Casscading Stylesheet((CSS) is used to format the layout of a webpage

With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!


Tips:The word cascading means that a style applied to a parent element will also apply to all children elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraphs, and other text elements within the body will also get the same color (unless you specify something else)!

Using CSS

CSS can be added to HTML documents in 3 ways

  • Inline CSS-by using style attribute inside HTML elements
  • Internal CSS-by using a style element inside the head section
  • External CSS-By linking an external page with css code

The most common way to add css, is keep the css file in an external file
