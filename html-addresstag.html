The HTML address tag defines the contact information for the author/owner of a document or an article.

The contact information can be an email address, URL, physical address, phone number, social media handle, etc.

The text in the address element will always render in Italic and browsers will always add a linebreak before and after the address element

Here is the way address element display in browsers

Name: Humayun Kabir
Phone: 01788636301;
House: 6/6, Motijheel, Dhaka
Occupasion: Software Engineer
